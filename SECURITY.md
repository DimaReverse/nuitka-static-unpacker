# Security Policy

## Reporting a vulnerability

If you find a security issue in this repository (for example, something that could be exploited in the tool itself, not in targets it analyzes), please **report it privately** before opening a public issue.

You can reach me via the GitHub "Report a vulnerability" button on the Security tab, or by opening a private issue.

## Scope

This tool is designed for static and dynamic analysis of software you are authorized to inspect. The maintainer is not responsible for misuse.

## Supported versions

The latest version in `main` is the only actively maintained version.
